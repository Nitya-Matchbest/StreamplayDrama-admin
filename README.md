# Drama Platform - Admin Panel

Admin panel for managing blog posts on Drama Platform with Crimson Cinema theme.

## Features
- Create new blog posts
- Edit existing posts
- Delete posts
- Upload images
- Search and filter blogs
- Category management
- Tag support

## Setup

1. Open `js/admin.js` and `js/manage.js`
2. Update the `API_BASE` constant with your backend URL:
   ```javascript
   const API_BASE = 'http://your-backend-url';
   ```

3. Open in browser:
   - Create Blog: `index.html`
   - Manage Blogs: `manage.html`

## Theme Colors

**Crimson Cinema Theme:**
- Background: #111827 (Midnight Navy)
- Primary: #DC2626 (Crimson Red)
- Secondary: #FFFFFF (Pure White)
- Accent: #F59E0B (Amber Gold)

## Usage

### Creating a Blog
1. Open `index.html`
2. Fill in the form fields
3. Upload a featured image
4. Add content sections
5. Add tags (optional)
6. Click "Create Blog Post"

### Managing Blogs
1. Open `manage.html`
2. Use search and filters to find blogs
3. Click "Edit" to modify a blog
4. Click "Delete" to remove a blog
